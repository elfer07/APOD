# Astro App
One of the most popular websites at NASA is the Astronomy Picture of the Day.

Application with a clean architecture this built with
 - MVVM
 - StateFlow
 - Coroutines
 - Retrofit
 - Navigation Components
 - ViewBinding
  
 # Support
 You need create a API key from https://api.nasa.gov/
